# Пример Volume Set (только индекс)

Этот пример показывает дескриптор `sitepack.volumes.json` для пакета, разделенного на несколько томов.
Фактические файлы томов не включены в репозиторий и должны создаваться инструментами экспортера.

Используйте этот пример для проверки схемы volume-set и для демонстрации формата метаданных поставки.
