# Использование материалов для LLM

Эта папка содержит краткую спецификацию и технические задания для генерации кода экспорта, импорта и проверки SitePack. Рекомендуемый порядок:

1. Прочитать `LLM_SPEC_SHORT.md` для понимания формата.
2. Выбрать задачу: `TASKS_VALIDATOR.md`, `TASKS_EXPORTER.md` или `TASKS_IMPORTER.md`.
3. Следовать требованиям к входным данным, выходным данным и отчетам.

Эти документы рассчитаны на прямую передачу LLM как технические briefs. Полные детали см. в `SPEC.md` и схемах в `schemas/`.
