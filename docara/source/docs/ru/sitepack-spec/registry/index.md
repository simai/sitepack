# Реестры

Раздел описывает справочные реестры, которые используются в спецификации SitePack для типов сущностей, форматов содержимого, медиа и преобразований.

Доступные реестры в английской версии:

- [Форматы содержимого](https://simai.github.io/sitepack/en/sitepack-spec/registry/content-formats)
- [Типы сущностей](https://simai.github.io/sitepack/en/sitepack-spec/registry/entity-types)
- [Типы медиа](https://simai.github.io/sitepack/en/sitepack-spec/registry/media-types)
- [Типы преобразований](https://simai.github.io/sitepack/en/sitepack-spec/registry/transform-kinds)
